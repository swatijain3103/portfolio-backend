For using swagger - first install swagger nuget package by using this command:
Install-Package Swashbuckle.AspNetCore
Then in startup.cs or in program.cs file edit ConfigureServices method:
services.AddSwaggerGen();
Now edit configure method type below details:
app.UseSwagger();
                app.UseSwaggerUI(c =>
                {
                    c.SwaggerEndpoint("/swagger/v1/swagger.json", "Contact API V1");
                    c.RoutePrefix = "swagger";
                });
Now add CORS for successfully connecting backend with frontend 
Edit configureService:
services.AddCors(options =>
            {
                options.AddPolicy("AllowAngularDevClient", builder =>
                {
                    builder.WithOrigins("https://localhost:4200")
                           .AllowAnyHeader()
                           .AllowAnyMethod();
                });
            });
Now add this in configure method before authorization line:
app.UseCors("AllowAngularDevClient");
Now edit launchSettings.json file use swagger in these two places:
"profiles": {
    "IIS Express": {
      "commandName": "IISExpress",
      "launchBrowser": true,
      "launchUrl": "swagger",
      "environmentVariables": {
        "ASPNETCORE_ENVIRONMENT": "Development"
      }
    },
    "ContactAPI": {
      "commandName": "Project",
      "launchBrowser": true,
      "launchUrl": "swagger",
      "applicationUrl": "https://localhost:44337;http://localhost:5000",
      "environmentVariables": {
        "ASPNETCORE_ENVIRONMENT": "Development"
      }
    }
